Forecast Mod Manager

Mod manager for Risk of Rain 2

Install Now on Windows 10 View on Thunderstore

Forecast is a fast, secure, and easy-to-use mod manager for Risk of Rain 2 mods. It allows you to quickly search through the thunderstore and install any mods with all of their dependencies. It also has a built-in config editor and profile-sharing system to tweak the mods and share profiles with friends using a 5 digit code.
