Privacy Policy for ethanbrews.me

The site and it's developer take privacy seriously. The site is hosted using firebase and uses certain firebase apis for the operation of the site and analytics. The firebase privacy policy can be found here.

The analytics data is used in order to better understand which features are being used by users to be able to make better development decisions to improve the website.

All data collected by the website is anonymous. If you choose to, you may reveal an identifier to the developer that would help diagnose a specific issue, but otherwise the data is not linked to you.
